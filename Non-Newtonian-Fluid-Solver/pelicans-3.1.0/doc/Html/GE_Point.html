SUMMARY :: METHOD
void set_as_barycenter(double alpha,
GE_Point const* ptA,
GE_Point const* ptB ) 
Set self as the barycenter of (ptA,1-alpha) and
(ptB,alpha).

Precondition
  • ptA != 0
  • nb_coordinates() == ptA->nb_coordinates()
  • ptB != 0
  • nb_coordinates() == ptB->nb_coordinates()

Postcondition
  • FORALL( i = 0, i < nb_coordinates(), ++i )
     FORMAL( coordinate( i ) == (1.0 - alpha) * ptA->coordinate( i ) + alpha * ptB->coordinate( i ) )
