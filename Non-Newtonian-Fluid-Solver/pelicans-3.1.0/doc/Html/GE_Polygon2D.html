  • result->dimension() == 2

SUMMARY :: METHOD
static GE_Polygon2D* create(PEL_Object* a_owner,
doubleVector const& coordinate_table ) 
Create and return an instance from the vector of double coordinate_table.

Precondition
  • coordinate_table.size() % 2 == 0

Postcondition
  • result != 0
  • result->owner() == a_owner
  • result->dimension() == 2
  • result->size() == coordinate_table.size() / 2
  • FORALL( i = 0, i < result->size(), ++i )
     result->vertex( i )->owner() == result && FORALL( j = 0, j < 2, ++j )
      result->vertex( i )->coordinate( j ) == coordinate_table( 2 * i + j )
  • !result->is_set_point_point_intersector()
  • !result->is_set_point_segment_intersector()
  • !result->is_set_segment_segment_intersector()

void insert_vertex_at(size_t i,
GE_Point const* pt ) 
Insert point pt has a new vertex of index i in the chain.
The function update is called.

Precondition
  • i <= size()
  • pt != 0
  • pt->nb_coordinates() == dimension()
  • is_set_point_point_intersector() && size() > 0 && i != size() ==> !point_point_intersector()->points_are_close( pt, previous_vertex( i ) )
  • is_set_point_point_intersector() && size() > 0 && i != size() ==> !point_point_intersector()->points_are_close( pt, vertex( i ) )
  • is_set_point_point_intersector() && size() > 0 && i == size() ==> !point_point_intersector()->points_are_close( pt, previous_vertex( size() - 1 ) )
  • is_set_point_point_intersector() && size() > 0 && i == size() ==> !point_point_intersector()->points_are_close( pt, vertex( 0 ) )

Postcondition
  • size() == OLD( size ) + 1
  • vertex( i ) == pt
