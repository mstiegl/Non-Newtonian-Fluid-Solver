Quadrature rules constructed by gathering quadrature rules on 
the subtriangles of the P1isoP2 reference element.

These 4 subtriangles are obtained by connecting the midpoints of the edges.

GE_QuadratureRule const* tria_rule ) 
Create and return an instance defined by setting a quadrature rule
based on tria_rule on each of the 8 subtriangles
subdividing the reference triangle.

Precondition
  • tria_rule->reference_polyhedron() == GE_ReferenceTriangle::object()

Postcondition
  • result != 0
  • result->owner() == PEL_Root::object()
