static GE_Vector* create(PEL_Object* a_owner,
GE_Point const* end,
GE_Point const* start ) 
Create a vector form start to 'end'.

Precondition
  • end != 0 && start != 0
  • end->nb_coordinates() == start->nb_coordinates()

Postcondition
  • result != 0
  • result->owner() == a_owner
  • result->nb_components() == start->nb_coordinates() && result->nb_components() == end->nb_coordinates()
  • FORALL( i = 0, i < result->nb_components(), ++i )
     FORMAL( result->component( i ) == end->coordinate( i ) - start->coordinate( i ) )
