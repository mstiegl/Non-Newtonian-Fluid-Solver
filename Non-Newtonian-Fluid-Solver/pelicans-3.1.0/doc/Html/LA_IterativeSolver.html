SUMMARY :: METHOD
void solve(LA_Matrix const* A,
LA_Vector const* b,
LA_Preconditioner* prec,
bool zero_initial_guess,
LA_Vector* x ) 
Solve the system defined by A as the LHS matrix and by b as the
RHS, and copy the solution into x ; use preconditioner prec. If
zero_initial_guess is false, use incoming x as the initial
guess.

Precondition
  • A != 0
  • b != 0
  • prec != 0
  • x != 0
  • A->nb_rows() == A->nb_cols()
  • A->nb_rows() == b->nb_rows()
  • A->row_distribution()->is_compatible( b->row_distribution() )
  • A->nb_cols() == x->nb_rows()
  • A->col_distribution()->is_compatible( x->row_distribution() )
  • prec->dimension() == A->nb_rows()
