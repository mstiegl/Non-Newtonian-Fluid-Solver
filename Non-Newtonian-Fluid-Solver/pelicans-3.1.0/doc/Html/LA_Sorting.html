Sort values in sortedValues with insert method.
rank is the corresponding permutation.
This sorting method is stable.

Precondition
  • values->nb_rows() > 0
  • sortedValues->nb_rows() == values->nb_rows()

Postcondition
  • FORALL( i = 0, i < values->nb_rows(), ++i )
     sortedValues->item( rank( i ) ) == values->item( i )
  • FORALL( i = 0, i < values->nb_rows() - 1, ++i )
     sortedValues->item( i ) <= sortedValues->item( i + 1 )

Sort values in sortedValues with insert method.
rank is the corresponding permutation.
This sorting method is stable.

Precondition
  • values.size() > 0
  • sortedValues.size() == values.size()

Postcondition
  • FORALL( i = 0, i < values.size(), ++i )
     sortedValues( rank( i ) ) == values( i )
  • FORALL( i = 0, i < values.size() - 1, ++i )
     sortedValues( i ) <= sortedValues( i + 1 )

static void quick_sort(LA_SeqVector* sortedValues,
size_t start,
size_t end ) 
Sort sortedValues in place with quick sort method.
This sorting method is not stable.

Precondition
  • sortedValues != 0
  • start <= end

Postcondition
  • FORALL( i = start, i < end - 1, ++i )
     sortedValues->item( i ) <= sortedValues->item( i + 1 )

static void column_rank(LA_SeqMatrix const* mat,
size_t_array2D& rank ) 
Fill in  matrix representing rank of items of mat in
their respective column using sort method.

Precondition
  • mat != 0
  • mat->nb_rows() > 0
  • mat->nb_cols() > 0

Postcondition
  • rank.index_bound( 0 ) == mat->nb_rows()
  • rank.index_bound( 1 ) == mat->nb_cols()
  • FORALL( j = 0, j < mat->nb_cols(), ++j )
     FORALL( i = 0, i < mat->nb_rows() - 1, ++i )
      rank( i, j ) <= rank( i + 1, j ) ==> mat->item( i, j ) <= mat->item( i + 1, j )
