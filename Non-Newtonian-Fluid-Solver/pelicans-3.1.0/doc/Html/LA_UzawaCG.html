LinearAlgebra /
Class LA_UzawaCG

  • PEL_Object
    • LA_TwoBlocksMethod
      • LA_UzawaCG
Iterative solvers for block linear systems of the form

       |  A   tB  |  | U |   | F |
       |          |  |   | = |   |
       |  B    C  |  | P |   | G |

using a preconditioned conjugate gradient version of the Uzawa algorithm.

reference : J. Cahouet, J.-P. Chabard
            Some Fast 3D Finite Element Solvers for the Generalized
            Stokes Problem
            Int. J. for Numer. Methods Fluids, Vol.8, 869-895 (1988)
