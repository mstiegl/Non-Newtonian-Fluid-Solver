Extension from PDE_FieldComposition SUMMARY :: METHOD
static PDE_FieldCompositionExp* create(PEL_Object* a_owner,
std::string const& a_name,
PEL_Data* expression,
PDE_SetOfDiscreteFields const* dfs ) 
Create and return an instance defined by expression. The variables
of expression should be of type PEL_DataDoubleVector. The
right part of their name should be the name of an item of dfs
or alternatively the name of another PDE_FieldCompositionExp (for
composition).

Precondition
  • expression != 0
  • expression->owner() == 0
  • dfs != 0

Postcondition
  • result != 0
  • result->owner() == a_owner
  • result->name() == a_name
  • expression->is_under_ownership_of( result )

void compute(void ) 
      Compute the value of all components of self, in two steps :
      The algorithm is :
         1. Compute the value of all components of the PDE_FieldComposition
            objects composed with self ;
         2. Call compute_self.

Precondition
  • has_consistent_internal_dependencies( this )
  • FORALL( start_variable_iterator(), valid_variable(), go_next_variable() )
     FORALL( ic = 0, ic < variable()->nb_components(), ++ic )
      variable_value_is_set( variable(), ic )
