Direct Known Derived Classes:
PEL_List, PEL_Vector
Collections whose items are in one-to-one correspondance with
integers of a contiguous interval with zero as lower bound.
items can be the NULL object.

virtual PEL_Sequence* create_clone(PEL_Object* a_owner ) const = 0
Create and a clone of self.

Postcondition
  • result != 0
  • result->owner() == a_owner
  • result->count() == count()
  • FORALL( it1 = create_iterator( 0 ), it2 = result->create_iterator( 0 ), it1->is_valid() && it2->is_valid(), it1->go_next(), it2->go_next() )
     it1->item()->has_same_address( it2->item() )
  • result->index_limit() == index_limit()
  • FORALL( i = 0, i < index_limit(), ++i )
     result->at( i ) == at( i )
