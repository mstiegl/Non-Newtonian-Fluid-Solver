NAME

cmp - requests forward to the PELICANS-based application ``pelcmp''

SYNOPSIS

pel cmp [-help|-man]

pel cmp [options...] file1 file2

DESCRIPTION

The PELICANS-based application ``pelcmp'' is called with the given options and arguments.

OPTIONS

-h, -help

Print a brief help message and exit.

-m, -man

Print the manual page and exit.

-v, -verbose

Execute verbosely.

-m, -modules mod

Limit the comparison to mod.

-d, -data datum

Limit the comparison to datum

ARGUMENTS

file1, file2

Name of the files containing containing the data to be compared.

EXAMPLE

pel cmp file_ref.pel file.pel

Compare the data of the files file_ref.pel and file.pel.
