NAME

newclass - text files creation for a new class

SYNOPSIS

pel newclass [-help|-man]

pel newclass [-path searchdir] model newclass

pel newclass [-pelicans] model newclass

DESCRIPTION

pel newclass creates the text files associated to a new class by copying those of a model class and replacing the name of the model class by the name of the new class.

The text files associated to the model class are assumed to have one of the following extensions : ``.hh'', ``.icc'' or ``.cc'' . They are searched in a set of paths defined by the calling options, or in the current directory if none.

OPTIONS

-h, -help

Print a brief help message and exit.

-m, -man

Print the manual page and exit.

-v, -verbose

Execute verbosely.

-path searchdir

Add directory searchdir to the list of paths that pel newclass will search for files model.hh, model.cc or model.icc . This option may be used any number of times.

-pelicans

Add all directories of the PELICANS repository defined by the environment variable PELICANSHOME to the list of paths that pel newclass will search for files model.hh, model.cc or model.icc .

ARGUMENTS

model

Name of the model class. The files model.hh, model.cc and model.icc will be searched, copied in the current directory if found, and any occurence of model will be replaced by newclass.

newclass

Name of the new class. The produced files will be newclass.hh, newclass.icc and newclass.cc, prodived that related files for the model class were found.

EXAMPLES

 pel newclass -pelicans GE_QuadratureRule_TEST MY_Class_TEST>
 pel   newclass -path ../src Model NewClass>
