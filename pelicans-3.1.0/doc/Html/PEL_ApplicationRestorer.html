Class PEL_ApplicationRestorer

  • PEL_Object
    • PEL_Application
      • PEL_ApplicationRestorer
Launchers of  executions of PEL_Application instances, starting from one
of their internal states that was previously saved on disk.

Given a PEL_Application instance whose internal state was saved
on disk using the facilities provided by PEL_ObjectWriter,
an object of PEL_ApplicationRestorer
   - first creates a PEL_Application instance from the original
     data deck,
   - then modifies its internal state according to one the savings performed
     during the original execution.
Thus, the newly created PEL_Application instance has a state identical
to that of the original PEL_Application instance at its execution point
where disk saving occured (with PEL_ObjectWriter). Calling
run for this new PEL_Application instance will subsequently produce
the same results as continuing the original execution from where it
was saved.
