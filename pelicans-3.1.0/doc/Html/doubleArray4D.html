Contruction of an instance whose all index intervals admit 0 as an
inclusive lower limit and respectively dim0, dim1, dim2 and
dim3 as an exclusive upper limit.

Postcondition
  • index_bound( 0 ) == dim0
  • index_bound( 1 ) == dim1
  • index_bound( 2 ) == dim2
  • index_bound( 3 ) == dim3
  • FORALL( i0 = 0, i0 < index_bound( 0 ), ++i0 )
     FORALL( i1 = 0, i1 < index_bound( 1 ), ++i1 )
      FORALL( i2 = 0, i2 < index_bound( 2 ), ++i2 )
       FORALL( i3 = 0, i3 < index_bound( 3 ), ++i3 )
        operator()( i0, i1, i2, i3 ) == val

SUMMARY :: METHOD
void raise_first_index_bound(size_t dim0,
double val = 0. ) 
Increase the exclusive upper limit of the first index interval
to dim0.

Precondition
  • dim0 > index_bound( 0 )
  • index_bound( 1 ) > 0 && index_bound( 2 ) > 0 && index_bound( 3 ) > 0

Postcondition
  • index_bound( 0 ) == dim0
  • FORALL( i0 = OLD( index_bound ), i0 < index_bound( 0 ), ++i0 )
     FORALL( i1 = 0, i1 < index_bound( 1 ), ++i1 )
      FORALL( i2 = 0, i2 < index_bound( 2 ), ++i2 )
       FORALL( i3 = 0, i3 < index_bound( 3 ), ++i3 )
        operator()( i0, i1, i2, i3 ) == val
