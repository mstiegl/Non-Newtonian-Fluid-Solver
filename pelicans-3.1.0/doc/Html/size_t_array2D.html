Contruction of an instance whose all index intervals admit 0 as an
inclusive lower limit and respectively dim0 and dim1 as an
exclusive upper limit.

Postcondition
  • index_bound( 0 ) == dim0
  • index_bound( 1 ) == dim1
  • FORALL( i0 = 0, i0 < index_bound( 0 ), ++i0 )
     FORALL( i1 = 0, i1 < index_bound( 1 ), ++i1 )
      operator()( i0, i1 ) == val

SUMMARY :: METHOD
void extract_section(size_t an_index,
size_t index_value,
size_t_vector& x ) const
Reinitialize x by copying all items of self whose an_index-th
index is equal to index-value.

Precondition
  • an_index < 2
  • index_value < index_bound( an_index )
  • an_index == 0 ==> x.size() == index_bound( 1 )
  • an_index == 1 ==> x.size() == index_bound( 0 )

Postcondition
  • an_index == 1 || FORALL( i1 = 0, i1 < index_bound( 1 ), ++i1 )
     x( i1 ) == operator()( index_value, i1 )
  • an_index == 0 || FORALL( i0 = 0, i0 < index_bound( 0 ), ++i0 )
     x( i0 ) == operator()( i0, index_value )

void raise_first_index_bound(size_t dim0,
size_t val = 0 ) 
Increase the exclusive upper limit of the first index interval
to dim0.

Precondition
  • dim0 > index_bound( 0 )
  • index_bound( 1 ) > 0

Postcondition
  • index_bound( 0 ) == dim0
  • FORALL( i0 = OLD( index_bound ), i0 < index_bound( 0 ), ++i0 )
     FORALL( i1 = 0, i1 < index_bound( 1 ), ++i1 )
      operator()( i0, i1 ) == val
