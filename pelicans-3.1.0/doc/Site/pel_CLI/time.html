NAME

time - management of a CPU time database for PELICANS-based applications

SYNOPSIS

pel time [-help|-man]

pel time [-verbose] -build_and_add file.db

pel time [-verbose] -merge target.db source.db

pel time [-verbose] -compare file1.db file2.db

pel time [-verbose] -print file.db

DESCRIPTION

pel time is devoted to the management of data bases collecting the execution time of PELICANS-based applications. This tool is used to manage execution timing database. These databases are built from result files produced by PELICANS-based applications.

OPTIONS

-h, -help

Print a brief help message and exit.

-m, -man

Print the manual page and exit.

-v, -verbose

Execute verbosely.

-build_and_add file.db

Search recursively from the current directory all files called resu, which are supposed to contain the messages associated to the run of a PELICANS-based application (collected with the pel -run utility). For each file resu, the execution time is read and added into file.db with the directory name as key (entries that already existed are replaced).

-merge target.db source.db

Add all entries of source.db into target.db (the entries that already exist in target.db are replaced).

-compare file1.db file2.db

Compare file1.db and file2.dbtwo database and print the differences.

-print file.db

Display the content of file.db.

EXAMPLES

pel time -build_and_add cpu.db

Build the database cpu.db from all the resu files of the subdirectories of the current directory.

pel time -print cpu.db

Display the content of cpu.db

pel time -compare cur.db ref.db

Compare cur.db and ref.db
