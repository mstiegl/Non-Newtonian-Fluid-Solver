Class LA_MatrixIterator

    • LA_MatrixIterator
Direct Known Derived Classes:
LA_BlockSeqMatrixIterator, LA_CRSmatrixIterator, LA_DenseMatrixIterator, LA_PelMatrixIterator, LA_ShiftedIndexMatrixIterator, LA_SymmetricMatrixIterator
Iterators to traverse a matrix or a single row of a matrix of type
LA_Matrix and visit all the stored coefficients exactly once.

The provided access to coefficients is efficient, sequential
and one-way.
