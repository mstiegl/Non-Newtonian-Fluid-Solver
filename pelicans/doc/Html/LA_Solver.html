SUMMARY :: METHOD
void set_initial_guess_nonzero(bool flg ) 
By default, iterative solvers assume an initial guess of zero by
zeroing the initial value of the solution vector that is passed
to solve.
If flg is false, this default is confirmed.
If flg is true, the solution vector that is passed to solve
is used as a possible nonzero initial guess.

Postcondition
  • is_iterative() ==> zero_initial_guess() == !flg

SUMMARY :: METHOD
void set_matrix(LA_Matrix const* mat ) 
Set the LHS matrix.
For direct solver, build inverse factorization of mat.
For iterative solvers, build preconditionner.

Precondition
  • mat != 0
  • mat->nb_rows() > 0
  • mat->nb_rows() == mat->nb_cols()
  • !matrix_is_set()
  • mat->is_synchronized()

Postcondition
  • !solution_is_achieved()
  • stop_on_error() ==> matrix_is_set()
  • matrix_is_set() ==> matrix_implementation() == mat->implementation()

SUMMARY :: METHOD
void solve(LA_Vector const* b,
LA_Vector* x ) 
Solve the system where b is the RHS, and return the solution
into x.

Precondition
  • matrix_is_set()
  • b != 0
  • b->nb_rows() == size()
  • b->implementation() == matrix_implementation()
  • b->is_synchronized()
  • b->row_distribution()->is_compatible( matrix()->row_distribution() )
  • x != 0
  • x->nb_rows() == size()
  • x->implementation() == matrix_implementation()
  • x->row_distribution()->is_compatible( matrix()->row_distribution() )

Postcondition
  • stop_on_error() ==> solution_is_achieved()
  • x->is_synchronized()
