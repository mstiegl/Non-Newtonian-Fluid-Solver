static PEL_ContextPair* create(PEL_Object* a_owner,
PEL_Context const* first,
PEL_Context const* second ) 
Create, initialize and return an instance (the result is the merge
of first and second context, second having priority).

Precondition
  • first == 0 <==> second == 0

Postcondition
  • result != 0
  • result->owner() == a_owner
  • first == 0 || second == 0 ==> result->nb_variables() == 0
  • FORALL( i = 0, first != 0 && i < first->nb_variables(), ++i )
     result->has_variable( first->variable( i ) )
  • FORALL( i = 0, second != 0 && i < second->nb_variables(), ++i )
     result->has_variable( second->variable( i ) )
  • FORALL( i = 0, i < result->nb_variables(), ++i )
     first->has_variable( result->variable( i ) ) || second->has_variable( result->variable( i ) )
  • FORALL( i = 0, i < result->nb_variables(), ++i )
     second->has_variable( result->variable( i ) ) ==> result->value( result->variable( i ) ) == second->value( result->variable( i ) )
  • FORALL( i = 0, i < result->nb_variables(), ++i )
     !second->has_variable( result->variable( i ) ) ==> first->has_variable( result->variable( i ) ) && result->value( result->variable( i ) ) == first->value( result->variable( i ) )
SUMMARY :: METHOD
void re_initialize(PEL_Context const* first,
PEL_Context const* second ) 
Reinitialize the internal state, as if the create method was
just completed (self is the merge of first and second context,
second having priority).

Precondition
  • first != 0 && second != 0
  • first != second
  • first != this
  • this != second

Postcondition
  • FORALL( i = 0, i < first->nb_variables(), ++i )
     has_variable( first->variable( i ) )
  • FORALL( i = 0, i < second->nb_variables(), ++i )
     has_variable( second->variable( i ) )
  • FORALL( i = 0, i < nb_variables(), ++i )
     first->has_variable( variable( i ) ) || second->has_variable( variable( i ) )
  • FORALL( i = 0, i < nb_variables(), ++i )
     second->has_variable( variable( i ) ) ==> value( variable( i ) ) == second->value( variable( i ) )
  • FORALL( i = 0, i < nb_variables(), ++i )
     !second->has_variable( variable( i ) ) ==> first->has_variable( variable( i ) ) && value( variable( i ) ) == first->value( variable( i ) )

virtual PEL_ContextPair* create_clone(PEL_Object* a_owner ) const
Create and a clone of self.

Provide new context pointing on self values and upper context.

Postcondition
  • result != 0
  • result->owner() == a_owner
  • result->nb_variables() == nb_variables()
  • FORALL( i = 0, i < nb_variables(), ++i )
     result->has_variable( variable( i ) ) && result->value( variable( i ) ) == value( variable( i ) )
