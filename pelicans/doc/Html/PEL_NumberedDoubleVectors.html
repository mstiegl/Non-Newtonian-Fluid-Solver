size_t a_size_of_items ) 
Create, initialize and return an instance devoted to store
doubleVector instances of size a_size_of_items.

Precondition
  • dbl_comp != 0

Postcondition
  • result != 0
  • result->owner() == a_owner
  • result->size_of_items() == a_size_of_items
  • result->nb_items() == 0
